# 个人作品集网站

这是一个基于 React + Vite + TypeScript 的个人作品集网站。

## 技术栈

- React 18
- Vite 5
- TypeScript
- Tailwind CSS 4
- Radix UI

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目已配置 Vercel 自动部署。所有配置文件已包含在仓库中。

## 项目结构

- `components/` - React 组件
- `assets/` - 静态资源文件
- `index.html` - HTML 入口文件
- `main.tsx` - React 入口文件
- `vite.config.ts` - Vite 配置
- `tsconfig.json` - TypeScript 配置
- `package.json` - 项目依赖
